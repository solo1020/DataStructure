# DataStrucure

#### Java 内存基础
> Java内存占用(字节)：
   * byte:1
   * short:2
   * int:4
   * long:8
   * float:4
   * double:8
   * boolean:1
   * char:2
   * 一个对象引用：8
   * 对象本身(除去对象内部存储的信息占用):16 保存对象头信息
   * 内存使用不够8字节 自动填充为8字节
   * 原始数据类型数组:24 (16字节为对象本身开销 + 4字节保存长度 + 4填充字节)

## 排序
### 冒泡排序
> 冒泡排序：  
每一轮找出 待比较元素组  中的 最大值放到该元素组最末尾   
具体实现：  
顾名思义，冒泡一样的一个个往上  
即 相邻的两个比较，将较大的一个放到后面 一直重复到完成该轮比较  
同时，待比较元素组 每轮都在变化  
因为 第一轮已经找到了原始数组中的最大值，不需要再进行比较  
所以 每一轮的 待比较数组为 0索引 到 (原始数组末尾索引 减 第几轮)
